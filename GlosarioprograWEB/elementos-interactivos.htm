¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web.

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño para dar estilo a páginas web.

Diálogos Modales (<dialog>)

<dialog id="miDialogo">
    <p>¿Estás seguro de continuar?</p>
    <button onclick="miDialogo.close()">Cerrar</button>
</dialog>

<button onclick="miDialogo.showModal()">
    Abrir diálogo
</button>
