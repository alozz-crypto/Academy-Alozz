Meta-Información en HTML

Metaetiquetas Esenciales

Charset Alta

Define la codificación de caracteres del documento:

<meta charset="UTF-8">

Viewport Alta

Configuración para dispositivos móviles:

<meta name="viewport" 
  content="width=device-width, initial-scale=1.0">

Description Alta

Descripción que aparece en los resultados de búsqueda:

<meta name="description" 
  content="Curso completo de HTML y CSS">

Metaetiquetas para SEO

Keywords Baja

Palabras clave (menos relevante hoy en día):

<meta name="keywords" 
  content="HTML, CSS, Web Development">

Author Media

Autor del documento:

<meta name="author" 
  content="Nombre del Autor">

Robots Media

Instrucciones para motores de búsqueda:

<meta name="robots" 
  content="index, follow">

Open Graph (Redes Sociales)

Título OG Alta

Título que se muestra cuando se comparte:

<meta property="og:title" 
  content="Título Impactante">

Descripción OG Alta

Descripción para redes sociales:

<meta property="og:description" 
  content="Descripción atractiva">

Imagen OG Alta

Imagen destacada al compartir:

<meta property="og:image" 
  content="imagen.png">

Enlaces Importantes

<!-- Favicon -->
<link rel="icon" href="favicon.ico" type="image/x-icon">

<!-- Hojas de estilo -->
<link rel="stylesheet" href="styles.css">

<!-- Precarga de recursos -->
<link rel="preload" href="fuente.woff2" as="font">

<!-- URL canónica -->
<link rel="canonical" href="https://www.ejemplo.com">

Favicon

Icono que aparece en la pestaña del navegador:

<link rel="icon">

Precarga

Mejora el rendimiento cargando recursos importantes primero:

<link rel="preload">
