Estructura del Documento HTML

Esqueleto Básico

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título del Documento</title>
</head>
<body>
    <!-- Contenido visible -->
</body>
</html>

Partes fundamentales:

  • DOCTYPE: Declara el tipo de documento
  • html: Elemento raíz con atributo de idioma
  • head: Metadatos y recursos no visibles
  • body: Contenido visible de la página

Zonas Semánticas

<header>Cabecera</header>
<nav>Navegación</nav>
<main>
    <article>Contenido principal</article>
</main>
<footer>Pie de página</footer>

Estructura típica:

<header>
<nav>
<main>
<article>
<footer>

Metadatos Esenciales

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Descripción del sitio">
    <link rel="icon" href="favicon.ico">
</head>

Importancia de cada uno:

  • charset: Define la codificación de caracteres
  • viewport: Hace el sitio responsive en móviles
  • description: Para SEO y resultados de búsqueda
  • favicon: Ícono visible en la pestaña del navegador
