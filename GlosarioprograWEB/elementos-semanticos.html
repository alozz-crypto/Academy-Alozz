Elementos Semánticos HTML5

¿Qué son los elementos semánticos?

Etiquetas que describen su significado tanto al navegador como al desarrollador:

  • Mejoran la accesibilidad
  • Optimizan el SEO
  • Estructuran el contenido claramente

Estructura Básica

<header>Cabecera</header>
<nav>Navegación</nav>
<main>
    <article>Contenido principal</article>
    <aside>Contenido secundario</aside>
</main>
<footer>Pie de página</footer>
<header> Cabecera del sitio </header>
<nav> Menú de navegación </nav>
<main>
<article> Artículo principal </article>
<section> Sección relacionada </section>
</main>
<aside> Barra lateral </aside>
<footer> Pie de página </footer>

Elementos Semánticos Comunes

Elemento Uso
<header> Cabecera de página o sección
<nav> Bloque de enlaces de navegación
<main> Contenido principal único
<article> Contenido autónomo (post, noticia)
<section> Agrupación temática de contenido
<aside> Contenido relacionado indirectamente
<footer> Pie de página o sección

Ejemplo Completo

<body>
    <header>
        <h1>Mi Sitio Web</h1>
        <nav>...</nav>
    </header>
    
    <main>
        <article>
            <h2>Artículo Principal</h2>
            <section>...</section>
        </article>
        <aside>Publicidad</aside>
    </main>
    
    <footer>Derechos reservados</footer>
</body>
